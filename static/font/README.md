# Graphik webfonts

Place your licensed Graphik webfont files here so the site uses Accenture’s primary typeface.

**Expected files (from your Accenture/Commercial Type webfont package):**

- `Graphik-Regular.woff2` (and optionally `.woff`)
- `Graphik-Medium.woff2` (optional; used for medium weight)
- `Graphik-Semibold.woff2` (and optionally `.woff`)
- `Graphik-Bold.woff2` (and optionally `.woff`)

If these files are missing, the site falls back to Inter. File names must match exactly (case-sensitive).
