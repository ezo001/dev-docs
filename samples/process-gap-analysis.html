Documentation process gap analysis

Process documentation · Case study · Sanitized for public portfolio

Problem

An organization migrated from a formal Document Development Life Cycle (DDLC) — SharePoint templates, named approvers, Planner tracking, release folders — to a faster Git-based web publishing pipeline. Teams needed to know what governance was lost and what to add back.

My role

I compared the legacy PDF process against the new job aid and repo implementation, documented gaps in a structured table, and wrote actionable suggestions per gap for doc leads and engineering managers.

Structure

  1. Summary table — old vs. new across output, source of truth, lifecycle, approval, tracking, metadata, maintenance
  2. Numbered gaps — each with old state, new state, gap description, and concrete suggestion
  3. Priority framing — which gaps block compliance vs. which are nice-to-have

Writing excerpt (sanitized)

Aspect Old process New process
Output PDF on internal developer hub Markdown/HTML on Azure Static Web App; optional DOCX export
Source of truth Word in SharePoint templates library Markdown in Git; DOCX is derived or conversion input
Approval Named approver; Teams Approvals workflow Optional PR review; no formal approver role

Example gap entry

Gap: No guaranteed sign-off by a named Approver before content goes live.
Suggestion: Add branch protection requiring PR review; document that the reviewer acts as Approver; optionally add approver metadata in front matter for reporting.

Outcomes

  • Doc leads used the analysis in a process workshop — 12 gaps prioritized in one session
  • Three suggestions adopted immediately: PR branch protection, "before you convert" checklist, review flag in SharePoint
  • Became the reference doc when onboarding new authors to the Git workflow

Why this belongs in a portfolio

Gap analyses demonstrate systems thinking: I don't just write steps, I map how documentation fits org process, tooling, and risk. Hiring managers for senior tech writer roles look for exactly this kind of artifact.
